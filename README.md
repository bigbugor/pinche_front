# 拼车微信小程序

信息发布型拼车平台

司机和乘客都需要实名认证，手机认证，为了保障双方的安全。

司机需要行驶证，驾驶证，身份证等信息的认证审核。


手机认证通过验证码核实

身份证，驾驶证，行驶证等证件需要上传照片后由后台管理员进行审核

审核过了司机和乘客才能发布拼车信息


进入拼车信息详情页

可以查看拼车的轨迹路线

可以复制微信号和拨打电话等方式进行联系
